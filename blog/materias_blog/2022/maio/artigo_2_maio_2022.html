ENTENDA SOBRE DADOS PESSOAIS E DADOS SENSÍVEIS

A LGPD não surgiu para proibir a utilização dos dados, mas sim para regulamentar a forma que eles poderiam ser utilizados. Para entender melhor sobre isso é preciso saber se os dados são pessoais, comuns ou dados sensíveis.

Os dados pessoais é a informação relacionada a pessoa identificada ou identificável. Assim, a LGPD traz um conceito amplo, pois quaisquer dados que possa permitir identificação de uma pessoa natural, pode ser considerado como dado pessoal.

OS DADOS PESSOAIS SÃO CAPAZES DE IDENTIFICAR UM INDIVÍDUO, E SÃO ELES:

A informação direta permite a imediata individualização da pessoa, já a indireta permite que por meio de diversas informações seja possível chegar à identificação da pessoa.

Já os dados sensíveis, eles precisam ser tratados de uma maneira mais criteriosa, pois não se trata apenas de dados numéricos, mas sim de dados que podem causar discriminação a uma pessoa, por conta disso exige uma proteção maior.

OS DADOS SENSÍVEIS SÃO MAIS DELICADOS, POIS PODE GERAR ALGUMA FORMA DE DESCRIMINAÇÃO, E SÃO ELES:

Diante disso, é determinado um rigor excessivo para tratar desses dados, por isso a implementação da LGPD na sua empresa é tão importante.
